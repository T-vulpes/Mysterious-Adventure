# Mysterious Adventure

Welcome to the Mysterious Adventure game! This is a text-based adventure game where you navigate through a forest, gather clues, and eventually escape to safety. The game is available in both English and Turkish.

## Table of Contents

- [Introduction](#introduction)
- [Installation](#installation)
- [Usage](#usage)
- [Gameplay](#gameplay)
- [Commands](#commands)
- [License](#license)

## Introduction

In this game, you find yourself in a dense forest with no memory of how you got there. Your goal is to navigate through the forest, gather clues, and find the correct password to escape. Along the way, you will encounter various challenges and characters that will help or hinder your progress.

## Installation

1. Clone the repository to your local machine:

```bash
git clone https://github.com/yourusername/mysterious-adventure.git
```

2. Navigate to the project directory:

```bash
cd mysterious-adventure
```

3. Open the `index.html` file in your preferred web browser to start the game.

## Usage

To start the game, open the `index.html` file in a web browser. Follow the on-screen instructions to navigate through the story and make choices.

## Gameplay

### Starting the Game

When you first open the game, you will see a welcome message with instructions on how to start the game. You can choose to start the game in English or Turkish by typing the appropriate command.

### Making Choices

As you progress through the story, you will be presented with various choices. Type the number corresponding to your choice and press Enter to proceed.

### Finding the Password

Throughout the game, you will gather pieces of the password. Once you have collected all parts of the password, you will be able to enter it to escape the forest.

## Commands

- `start en`: Start the game in English.
- `start tr`: Start the game in Turkish.
- `restart`: Restart the game from the beginning.
- `clear`: Clear the game screen.
- `help`: Display a list of available commands.
